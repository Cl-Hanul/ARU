## 소개
`아르(ARU)`는 디스코드 다 기능 봇입니다.
여러 기능을 가지고 있습니다.

모든 코드는 오픈소스이며, 라이선스만 지킨다면 누구든 사용 가능합니다.

## 사용법
1.  소스코드 다운 후, `launch_set.py` 실행
2.  각 api에 따라 발급 후 작성 ([api 리스트])
3. `launch.py`를 실행
